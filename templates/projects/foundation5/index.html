ASP.NET - Foundation 5 Framework!

Yo ASPNET-F5 build:

  • SASS/SCSS compiling (Ruby or Node/Libsass)
  • Build html templates in app folder
  • Copy files to wwwroot, includes & app directories
  • Includes directory for header.cshtml & footer.cshtml
  • Publishing to dist directory & includes directory
  • Server with LiveReload
  • JSHint
  • Grunt Bower install
  • Imagemin
  • Font Awesome (option)
  • Jade templating engine (option)

Grunt tasks:

  • 'grunt' - watching (Sass, Server on 127.0.0.1:9000 with LiveReload)
  • 'grunt compile-sass' - Sass
  • 'grunt compile-jade' - Jade
  • 'grunt validate-js' - JSHint
  • 'grunt bower-install' - injecting bower libraries
  • 'grunt copy-bower' - copy files to app/bower_components
  • 'grunt copy-app-files' - copy files to wwwroot & includes
  • 'grunt publish' - dist directory
  • 'grunt server-dist' - server on 127.0.0.1:9001 - dist directory (preview only)

KPM tasks:

  • 'kpm restore' - run install nuget and npm & bower packages
  • 'kpm build' - Build project files
  • 'k web' - Run kpm server on localhost:5001
  • 'k kestrel' - Run kpm server on localhost:5004
  • 'k gen' - Code generation
  • 'k ef' - Entity framework commands
